# Privacy Policy

AI Usage Tracker has **no telemetry**, advertising, analytics, crash-reporting service, or developer server. It does not sell or share personal information.

Authentication secrets are stored locally in Windows Credential Manager or read from the same local official CLI files used by the relevant provider. Preferences, non-secret configuration and state are stored locally in your Windows user profile.

The app contacts enabled providers directly to authenticate and retrieve usage information. It contacts GitHub to check for and download application updates. Those services receive ordinary network information such as your IP address and are governed by their own privacy policies. No usage data is routed through a server operated by the AI Usage Tracker developer.

Removing the app does not remove provider CLI files. See [SUPPORT.md](SUPPORT.md) for local-data cleanup guidance.
